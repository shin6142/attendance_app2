# attendance management application
## Description
This application automates recording monthly attendances. for Slack and Freee User.

## Stack
- Kotlin
- Spring Boot
- TypeScript
- React
- PostgreSQL
- jooq
- liquibase
- openapi-generator
- Docker

## set up
### initialize application & start application
```
$ make initialize
```

### start application
```
$ make start
```

### stop application
```
$ make stop
```
